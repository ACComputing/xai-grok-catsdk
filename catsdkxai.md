# CatSDK 0.1

CatSDK 0.1 is a lightweight, comfort-focused SDK philosophy for building warm, responsive, expressive tools, launchers, emulators, games, and UI systems.

It prioritizes cozy design, fast iteration, clean structure, friendly status messages, minimal dependencies, optional assets, readable code, and easy AI-assisted modification.

When generating CatSDK projects, keep them simple, stable, playful, dark-themed with blue accents when fitting, and always preserve the user's requested names, titles, and vibe.
